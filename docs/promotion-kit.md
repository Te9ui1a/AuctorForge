# AuctorForge Promotion Kit

This page collects short descriptions, launch copy, and community prompts for sharing AuctorForge.

## One-line Description

面向中文网文作者的本地优先 AI 长篇创作工作台。

English: A local-first AI creative workbench for Chinese long-form fiction and webnovel writers.

## Short Description

AuctorForge helps Chinese long-form fiction writers keep project files, story context, drafts, review notes, AI assistant chat, workflow guidance, and model settings in one Web UI.

It is designed around author control: local project files first, transparent model configuration, privacy-aware workflows, and reusable templates for characters, worldbuilding, outlines, revision, and continuity review.

## Who It Is For

- 中文网文作者、长篇连载作者、系列文作者
- 想用 AI 辅助创作，但不想把作品完全交给 AI 代写的人
- 需要管理人物、设定、大纲、正文、审稿意见和长篇一致性的作者
- 想共建中文长篇创作工作流、提示词模板和本地优先写作工具的开发者

## What Makes It Different

- Focused on Chinese long-form fiction and webnovel workflows.
- Local-first project files, so writers can inspect, back up, and move their work.
- AI is treated as an assistant for drafting support, review, and organization, not as an author replacement.
- Privacy and manuscript safety are first-class concerns in documentation and product direction.
- Built as an open-source Web UI that developers and writers can shape together.

# 新书预期 (Initial Expectation)

> 在正式动笔或拆解样板书之前，请花几分钟思考你对你将要创作的这本书的直觉预期。这不需要很精确，但能帮助你更好地掌控你的小说，而不是成为 AI 的傀儡。

## 1. 题材边界 (Genre Boundaries)
- **核心类型**：（例如：都市异能、玄幻升级、仙侠、科幻、悬疑、无限流...）
- **不想写什么**：（例如：不写后宫、不写系统、不写虐主、不写无脑反派...）
- **必须有什么**：（例如：强金手指、反转、团队羁绊、种田经营、扮猪吃虎...）
- **核心爽点**：（例如：低调反杀、信息差破局、资源滚雪球、压抑后的释放...）

## 2. 预期篇幅与主线 (Length & Main Arc)
- **预期字数**：(例如：200万字、500章...)
- **一句话主线**：（例如：角色从起点状态走向目标状态的故事）

## 3. 预期的设定、剧情与人设 (Plot & Characters)
- **主角形象**：（例如：苟、冷静、腹黑、嘴硬心软、底层逆袭、表面普通但关键时刻很狠...）
- **名场面脑洞**：(如果有想好的画面，可以写在这里)
- **预期设定**：（例如：金手指机制、力量体系、城市规则、门派/组织关系等）
- **预期剧情**：（例如：主角如何拿到第一桶金、如何付出代价、如何升级能力、如何反制强敌...）
- **预期人物**：(例如：主角、核心反派、师友、队友、情感线角色等)

## 4. 个人执念 (Personal Obsessions)
- **为什么想写这本书？**：（例如：为了表达某种情绪、回应某个长期兴趣...）
- **对标作品**：（例如：想接近某类作品的阅读体验...）

> 1. 任何 AI Agent 在读取本文件时，必须**立即读取** `.novelflow/skills/longformnovel/SKILL.md`，了解完整的技能包流程。
> 2. 后续所有创作步骤必须遵循技能包内定义的标准流程：
>    - **启动阶段**：Analyze（样板书分析）/ Define（从零定义）/ Guide（存量整合/灵活切入）
>    - **设定阶段**：Ideation（脑暴与设定）
>    - **规划阶段**：Outline（总纲 → 卷纲 → 章纲）
>    - **生产阶段**：Write（逐章写作）↔ Review（质检审稿）
> 3. **严禁脱离技能包流程**另行推进或自创步骤。
> 4. 不确定下一步时，参考本文件 §8（项目进度与待办），或主动询问用户调用哪个子技能。
